## 유스케이스 다이어그램

```mermaid
graph LR
  %% ── 액터 ──────────────────────────────────────────────
  Guest(["🧑\n비인증 사용자"])
  AuthUser(["🧑\n인증된 사용자"])

  %% ── 시스템 경계 ────────────────────────────────────────
  subgraph TodoListApp["  TodoListApp 시스템  "]

    subgraph AUTH["🔐 인증"]
      A01("UC-A-01\n회원가입")
      A02("UC-A-02\n로그인")
      A03("UC-A-03\n로그아웃")
    end

    subgraph USER["👤 사용자 관리"]
      U01("UC-U-01\n개인정보 수정")
    end

    subgraph TODO["📝 할일 관리"]
      T01("UC-T-01\n할일 등록")
      T02("UC-T-02\n할일 수정")
      T03("UC-T-03\n할일 삭제")
      T04("UC-T-04\n완료 처리")
      T05("UC-T-05\n할일 단건 조회")
    end

    subgraph CATEGORY["🏷️ 카테고리 관리"]
      C01("UC-C-01\n카테고리 추가")
      C02("UC-C-02\n카테고리 수정")
      C03("UC-C-03\n카테고리 삭제")
    end

    subgraph FILTER["🔍 조회 및 필터링"]
      F01("UC-F-01\n할일 목록 조회")
      F02("UC-F-02\n카테고리별 필터링")
      F03("UC-F-03\n기간 필터링")
      F04("UC-F-04\n완료 여부 필터링")
      F05("UC-F-05\n복합 필터링")
    end

  end

  %% ── 비인증 사용자 연결 ─────────────────────────────────
  Guest --> A01
  Guest --> A02

  %% ── 인증된 사용자 연결 ─────────────────────────────────
  AuthUser --> A03
  AuthUser --> U01
  AuthUser --> T01
  AuthUser --> T02
  AuthUser --> T03
  AuthUser --> T04
  AuthUser --> T05
  AuthUser --> C01
  AuthUser --> C02
  AuthUser --> C03
  AuthUser --> F01
  AuthUser --> F02
  AuthUser --> F03
  AuthUser --> F04
  AuthUser --> F05

  %% ── include 관계 ───────────────────────────────────────
  T02 -. "«include»" .-> T05
  F02 -. "«include»" .-> F01
  F03 -. "«include»" .-> F01
  F04 -. "«include»" .-> F01
  F05 -. "«include»" .-> F01

  %% ── extend 관계 ────────────────────────────────────────
  A01 -. "«extend»\n회원가입 성공 시\n기본 카테고리 자동 생성" .-> C01
```

## 관계 설명

| 관계 유형 | From | To | 설명 |
|-----------|------|-----|------|
| «include» | UC-T-02 할일 수정 | UC-T-05 할일 단건 조회 | 수정 화면 진입 시 반드시 단건 조회 선행 |
| «include» | UC-F-02 카테고리별 필터링 | UC-F-01 할일 목록 조회 | 필터링은 목록 조회를 포함 |
| «include» | UC-F-03 기간 필터링 | UC-F-01 할일 목록 조회 | 필터링은 목록 조회를 포함 |
| «include» | UC-F-04 완료 여부 필터링 | UC-F-01 할일 목록 조회 | 필터링은 목록 조회를 포함 |
| «include» | UC-F-05 복합 필터링 | UC-F-01 할일 목록 조회 | 복합 필터링은 목록 조회를 포함 |
| «extend» | UC-A-01 회원가입 | UC-C-01 카테고리 추가 | 회원가입 성공 시 기본 카테고리(일반·업무·개인) 자동 생성 |
